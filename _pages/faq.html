Below are some of the most frequently asked questions (FAQ) about {{ site.application_name_full }}.

{% capture question %} What is {{ site.application_name_full }}? {% endcapture %} {% capture answer %}

{{ site.application_name_full }} is a mobile application designed for Apple iPhones and Apple iPads. {{ site.application_name_full }} can be used to secure your online identity. You already have a username and password for various online services. However, your password can be breached, which makes you vulnerable to hacks. To mitigate this risk, security experts came up with a solution called Multi-Factor Authentication (MFA).

MFA comes in many forms, the most common one being a 6-digit code on your phone. Whenever you authenticate to one of your accounts, you have to enter your 6-digit code besides your username and password. The 6-digit code changes every 30 seconds, which is why it can't be easily breached. It increases the cyber resilience of your online identity.

You can enable MFA in almost every online service. It is generally found on the (security) settings page of the online service. Whenever you enable it, you need to scan a QR-code using {{ site.application_name_full }}. The 6-digit code that changes every 30 seconds is then stored in {{ site.application_name_full }}.

After enabling MFA for an online service, you have to enter the 6-digit code everytime you authenticate using your username and password. You can find the 6-digit code in {{ site.application_name_full }} at all time.

{% endcapture %} {% include question_capture.html category='General' question=question answer=answer %} {% capture question %} What are one-time passwords? {% endcapture %} {% capture answer %}

A one-time password (OTP) is a password that you can have in addition to the username and password you use for online services. As the name states, you can only use an OTP once. Don't worry, the OTP will change every 30 seconds, after which you can use it again. As it changes every 30 seconds, it is difficult for hackers to obtain it. Therefore, using an OTP increases your cyber resilience.

{% endcapture %} {% include question_capture.html category='General' question=question answer=answer %} {% capture question %} Are my one-time passwords encrypted? {% endcapture %} {% capture answer %}

Yes, all one-time passwords (effectively the seeds) stored in {{ site.application_name_full }} for iOS are encrypted using your encryption (master) password and your PIN code. If you have Apple iCloud synchronisation enabled, your one-time passwords in Apple iCloud are encrypted using your encryption (master) password.

{{ site.application_name_full }} for MacOS does not store any one-time passwords. It only receives them from {{ site.application_name_full }} for iOS. The connection between {{ site.application_name_full }} for iOS and {{ site.application_name_full }} for MacOS is encrypted using an encryption key that was generated during the installation of {{ site.application_name_full }} for MacOS.

If you export a ZIP-archive using {{ site.application_name_full }} for iOS, the ZIP-archive is encrypted using your encryption (master) password.

{% endcapture %} {% include question_capture.html category='General' question=question answer=answer %} {% capture question %} Is {{ site.application_name_full }} available for other operating systems? {% endcapture %} {% capture answer %}

{{ site.application_name_full }} is only available for iOS and MacOS. It will not be supported on other operating systems such as Windows or Android. The {{ site.application_name_full }} MacOS application is simply a companion application that receives OTP's when you tap them in {{ site.application_name_full }} for iOS. The MacOS application will never evolve in a fully fledged application as it compromises the security level.

{% include icon.html name='download' %} Download {{ site.application_name_full }} beta {% endcapture %} {% include question_capture.html category='Installation' question=question answer=answer %} {% capture question %} How do I configure {{ site.application_name_full }} for iOS once installed? {% endcapture %} {% capture answer %}
  1. Open {{ site.application_name_full }} from your homescreen.
    It is the white icon with the red shield.
  2. Select a synchronisation provider. Afterwards tap continue.
    This provider will store your passwords (encrypted) and synchronise them across your devices. We strongly recommend using Apple iCloud.
  3. Choose a strong encryption (master) password and save it in your password manager. Afterwards tap continue.
    If you forget this encryption (master) password you will lose access to your OTPs. You don't have to enter it every time you open the app, just during (re)installation.
  4. You need to set a PIN code. Afterwards tap continue, confirm your PIN code, and tap continue again.
    The PIN code is required everytime you unlock the app.
  5. At last, {{ site.application_name_full }} gives you the option to enable biometrical unlock (e.g. FaceID).
    By enabling biometrical unlock you don't have to enter your PIN code everytime you open the app.
{% endcapture %} {% include question_capture.html category='Installation' question=question answer=answer %} {% capture question %} How do I configure {{ site.application_name_full }} for MacOS once installed? {% endcapture %} {% capture answer %}
  1. A prereqesuit for using the MacOS app is that you have {{ site.application_name_full }} for iOS installed.
    You cannot use the MacOS app without {{ site.application_name_full }} for iOS.
  2. After installation, click on the {{ site.application_name_full }} shield in the MacoS status bar.
    The status bar is the bar in the top of your screen (it contains e.g. your battery level).
  3. Open preferences via the {{ site.application_name_full }} menu in the status bar.
    An actual application window will open after clicking preferences.
  4. Click on 'Get started' and in the mean time open {{ site.application_name_full }} on your iOS device.
    We will use the iOS app to connect it to the MacOS app.
  5. In the iOS app, go to 'Settings', then 'Receivers' and tap the '+' button in the top right corner.
    A camera view will now open which you need to point to your MacOS device.
  6. Scan the QR-code in the MacOS app using the iOS app.
    {{ site.application_name_full }} for MacOS and iOS are now synced. Whenever you tap an OTP in {{ site.application_name_full }} for iOS, the OTP will be sent to the clipboard of your MacOS device.

{% endcapture %} {% include question_capture.html category='Usage' question=question answer=answer %} {% capture question %} How do I extract a ZIP-archive export? {% endcapture %} {% capture answer %}

Make sure to send the export file to your computer. On your computer, you can extract the ZIP-archive using any ZIP-utility. Your ZIP-archive export is encrypted using your encryption (master) password that you've configured during the setup of {{ site.application_name_full }} for iOS.

As an alternative way to restore from your ZIP-archive export, you can scan each QR-code in the HTML file of your ZIP-archive export manually.
